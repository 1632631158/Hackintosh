### 我的配置  
系统 10.14.5(18F132)  
CPU I7-8700K  
主板 MSI Z370 Krait Gaming (银环蛇)  
内存 CORSAIR 海盗船 DDR4 3000  
显卡 蓝宝石 RX580 超白8G  
SMBIOS 19.1  
引导为OC 版本号0.54  

## 特别注意  
需在BIOS关闭CFG LOCK（MSR 0xe2寄存器）默认开启   
需自己生成SMBIOS，已经抹掉SN  
非银环蛇主板使用此EFI很可能会卡开机
